Our AWS Services

Amazon Elastic Compute Cloud (Amazon EC2) delivers scalable computing capability in the Amazon Web Services (AWS) Cloud.

Lambda is a compute service that lets you build applications that respond quickly to new information and events.

AWS Batch helps you to run batch computing workloads on the AWS Cloud.

Amazon Transcribe is an ASR service that allows developers to easily integrate speech-to-text functionality to their apps.

A machine learning platform with integrated development, training, deployment, and analysis capabilities.

For creating speech and text-based conversational user interfaces for apps. AWS offers Amazon Lex.

Amazon Simple Storage Service (Amazon S3) is an object storage service that provides industry-leading scalability, data availability, security, and performance.

Amazon DynamoDB is a fully managed, serverless, key-valued NoSQL database designed to run high-performance applications of any size.

Amazon Elastic Block Store (Amazon EBS) is an easy-to-use, scalable, and high-performance block storage service designed for Amazon Elastic Compute Cloud (Amazon EC2).
